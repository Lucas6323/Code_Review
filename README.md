**Objetivo do Projeto**

Este projeto tem como objetivo demonstrar, na prática, o funcionamento do fluxo de colaboração no GitHub, utilizando Branches, Commits, Push e Pull Requests para desenvolvimento em equipe.

Cada integrante do grupo desenvolve sua própria parte do projeto em uma branch individual, permitindo que as alterações sejam revisadas antes de serem integradas ao código principal.

**Estrutura do Projeto**

O projeto consiste em um site de portfólio, onde cada integrante possui sua própria página dentro do site.

A estrutura básica inclui:

Página principal do site

Página individual de cada integrante

Estilização utilizando HTML e CSS

Integração das páginas através do repositório colaborativo
